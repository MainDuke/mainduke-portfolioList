이벤트 : 마우스나 키보드로 입력할 때 발생하는 신호
	이벤트 핸들러(함수) : 이벤트가 발생했을때 수행되는 함수를 말한다.
	
	onClick : 마우스를 클릭했을때
	onMouseOver : 마우스 커서가 올려졌을때
	onLoad :  html page 시작시에 실행 (body 태그에서만 사용되어야 한다.)
	onUnload : page 끝 page에서 나갔을때 (body 태그에서만 사용되어야 한다.)
	onChange : 값이 변경되었을때
	onBlur :  커서가 있을 시 (focus 을 잃었을때) 발생.
	onFocus :  커서가 있을 시 (focus 을 얻었을 때) 발생.
	
	onKeyDown : 키를 눌렀을 때 발생.
	onKeyUp : 키를 눌렀다가 떼었을떄 발생하는 핸들러

노트 정리 <form> 에서 사용하는 태그 정리 </form> <form methud="get/post" action="서버주소" encType="인코딩문자" target="문서이름"> </form> <form methud="get/post" action="aa.jsp" encType="인코딩문자(파일 업로드 할때 사용)" target="aaa"> </form> methud : http 명령을 기술한다, 보통은 get과 post 중 하나를 기술한다 생략하면 get 방식이다. action : form의 내용을 웹 서버에 전송할 때 처리 될 CGI 프로그램이나 서블릿(JSP)프로그램의 URL이나 이름을 기술한다. action ="aa.jsp" action = "http://localhost:9001/프로젝트이름(웹어플리케이션)/중간경로(폴더)/aa.jsp" encType : 인코딩 타입을 기술한다. 파일 업로드 하기 위해서는 encType="multipart/form-data" 값을 기술해야한다. target : CGI 프로그램이나 서블릭 프로그램 데이터를 처리한 후 결과를 보여줄 프레임이나 창 이름을 기술한다. ★form 내부에서 쓰여지는 tag 들 button : 버튼을 클릭하면 명령 수행을 한다. cherkbox : 여러개의 선택 항목 중 하나를 선택할 수도 있고, 여러개를 선택할 수도 있고, 선택 안 할 수도 있다. radio : 여러개 중에 오직 하나만 선택할 수 있다. text : 텍스트를 입력 할 때 사용 password : 패스워드를 입력할 때 사용 file : 클라이언트 파일을 서버로 업로드 할 때 사용 예> 메일 보내기 에서, 첨부 파일 하는 것 submit : 입력 데이터를 서버로 전송 예시 : <input type = "입력타입 기술" name =" 이름(자바변수와 유사) value="값"> type : 입력 형태를 기술한다 name : 현재 inputd 의 태그이름을 기술한다. (중요함 , 자바변수와 같은 기능) value : input 태크 값 기술 (변수 =값), 여기서는 값을 서버로 보낸다. 예: <input type = "text" name =" id" value="kim"><!-- type - "text" : 문자 한 줄 입력 --> <input type = "password" name =" pwd"value="12345"><!-- 암호 입력 --> <input type = "cherkbox" name =" hobby" value="트래킹"> <input type = "radio" name =" gender" value="남자"> <input type = "hidden" name =" num" value="3"><!-- 값을 숨겨서 보낼때 --> <input type = "bottom" name =" " value="자바스크립트함수 호출"><!-- 버튼을 클릭해서 명령 수행 --> <input type = "submit" name =" " value="서버로 전송"><!-- form에서 기술한 애용을 웹 서버로 전송--> <input type = "reset" name =" " value="입력한 데이터 취소"><!-- form에 입력한 데이터를 모두 지울때-->
